agic

A static site generator powered by Deno + React

Get StartedDemos

Easy to configure

  • Convention over configuration
  • Single config file pagic.config.ts
  • Intuitive design

Support md and tsx

  • Render md/tsx to static HTML page
  • Support React Hooks
  • Pre-render to static HTML, run as an SPA once loaded

Themes and plugins

  • Official themes default/docs/blog with dark mode
  • Combine plugins to build process
  • Import third-party themes or plugins through URL

Get up and running in seconds

# Install pagic
deno install --allow-read --allow-write --allow-net --allow-env --allow-run --name=pagic https://deno.land/x/pagic/mod.ts

# Create pagic.config.ts and README.md
mkdir site && cd site && echo "export default {};" > pagic.config.ts && echo "# Hello world" > README.md

# Run pagic
pagic build --watch --serve
